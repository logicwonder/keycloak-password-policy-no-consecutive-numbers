# Keycloak Password Policy - No Consequtive numbers

The password shall not have consequtive numeric sequence like 12, 23, 123 etc.

## Configuration
1. Copy the jar to keycloak/providers/ folder
2. Run command
   `bin/kc.sh build`
4. Start Keycloak server
   `bin/kc.sh start`
